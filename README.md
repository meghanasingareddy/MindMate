# MindMate

MindMate is an AI-powered mental wellness companion designed to provide a safe, judgment-free space for users to express their thoughts, track emotions, and receive supportive guidance.

---

## Why MindMate?

Mental health support is often inaccessible, expensive, or uncomfortable to seek. MindMate addresses this by offering:

* A private space to talk anytime
* Emotion-aware AI responses
* Consistent daily mental check-ins
* Lightweight and approachable interface

---

## Features

* AI-powered conversational support
* Sentiment-aware responses
* Daily mood check-ins
* Clean and calming UI
* Always available (24/7 support simulation)

---

## Tech Stack

* React (Vite)
* TypeScript
* Tailwind CSS
* shadcn/ui

---

## Getting Started

### Prerequisites

* Node.js (v16+ recommended)
* npm

### Installation

```bash
git clone https://github.com/meghanasingareddy/MindMate.git
cd MindMate
npm install
npm run dev
```
